# 設計指針監査チェックリスト (For Architecture)

## 1. 原則との整合性 (Principles)
- [ ] **クリーンアーキテクチャ:** 依存性のルールや責務の分離に違反していないか？
- [ ] **進化性:** 将来の変更を不当に制限するような決定（早すぎる最適化など）が含まれていないか？

## 2. トレードオフとリスク (Trade-offs & Risks)
- [ ] **4大リスク評価:** 価値、ユーザビリティ、実現可能性、生存性の観点でリスクが洗い出されているか？
- [ ] **対案の検討:** 「採用しなかった案」とその理由が考慮されているか？

## 3. 明瞭性 (Clarity)
- [ ] **コンテキスト:** 「なぜこの決定が必要なのか」という背景が明確か？
- [ ] **論点:** 議論すべきポイントが絞り込まれているか？
