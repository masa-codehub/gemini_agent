# gemini_agent

Gemini CLI エージェントおよび関連エコシステムの共通テンプレートリポジトリです。

## 概要

このリポジトリは、Gemini CLI を活用した開発フローの標準化、共通スクリプト、および GitHub Actions ワークフローのテンプレートを提供します。

実質的なアプリケーションロジックはこのリポジトリ自体には含まれず、主に以下の役割を担います。

- **共通テンプレートの提供:** 各プロジェクトリポジトリのベースとなる設定やディレクトリ構造を管理します。
- **テンプレート同期の起点:** 各プロジェクトリポジトリで改善されたワークフローやスクリプトが、このリポジトリにプルリクエストとして集約（バックポート）され、他のプロジェクトへ展開されるハブとなります。

## 主要な構成

- `.gemini/`: Gemini CLI の動作を規定するコア設定とスキルセット。
- `.github/workflows/`: CI/CD、同期、自動化タスクなどの GitHub Actions 定義。
- `.build/`: 環境セットアップおよび実行用スクリプト。
- `reqs/`: デザインドキュメントやタスク管理の標準構成。

## メンテナンス方針

各プロジェクトからのテンプレート部分の改善提案は、このリポジトリへのプルリクエストとして受け付けます。承認された変更は、自動同期ワークフローを通じて他の各プロジェクトリポジトリへと展開されます。
